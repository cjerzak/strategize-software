# Deprecated One-Step Estimator Archive

This directory preserves the removed one-step estimator implementation and its
dedicated tests.

- Status: deprecated and no longer part of the active `strategize` package API
- Purpose: reference, recovery, and historical inspection only
- Active package surface: `strategize_onestep()` is no longer exported

Contents:

- `R/`: archived one-step implementation files
- `man/`: archived one-step man page
- `tests/testthat/`: archived one-step tests and one-step-specific helpers
